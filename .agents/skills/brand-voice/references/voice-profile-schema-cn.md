# Voice Profile Schema

构建可复用 voice profile 时请严格使用以下结构：

```text
VOICE PROFILE
=============
作者：
目标：
置信度：

来源集合
- 来源 1
- 来源 2
- 来源 3

节奏
- 关于句长、pacing 和碎片化的简短说明

内容密度
- 写作内容的密集程度或解释性强弱

大小写使用规范
- 常规、混合或视场景而定

括号使用规范
- 括号的适用场景和禁用场景

问题使用规范
- 极少使用、频繁使用、反问、直接提问或基本不使用

主张表述风格
- 如何构建、支撑和强化主张

偏好表达手法
- 作者常用的具体表达手法

禁用表达手法
- 作者明确不使用的特定模式

CTA 规则
- 如何、何时或者是否需要以请求收尾

渠道注意事项
- X：
- LinkedIn：
- Email：
```

指导原则：

- 保持 profile 具体明确，有来源支撑。
- 使用简短列表项，不要使用散文式段落。
- 每一项禁用表达手法都应能在来源集合中观察到，或由用户明确要求。
- 如果来源集合存在冲突，明确指出分歧，不要模糊折衷成毫无特点的内容。

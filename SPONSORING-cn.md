# 赞助 ECC

ECC 是一个开源的 agent 运行时性能系统，支持 Claude Code、Cursor、OpenCode 和 Codex 等应用/命令行工具。

## 为什么赞助

赞助将直接用于：

- 更快的 bug 修复和发布周期
- 跨运行时平台的功能一致性工作
- 面向社区免费提供的公开文档、skills 和可靠性工具

## 赞助层级

以下是基础的赞助方案，可以根据合作范围进行调整。

| 层级 | 价格 | 适用对象 | 包含权益 |
|------|-------|----------|----------|
| 试点合作伙伴 | $200/月 | 首次赞助合作 | 月度指标更新、路线图预览、优先维护者反馈 |
| 成长合作伙伴 | $500/月 | 正在积极采用 ECC 的团队 | 试点权益 + 月度办公时间同步 + 工作流集成指导 |
| 战略合作伙伴 | $1,000+/月 | 平台/生态系统合作 | 成长权益 + 协同发布支持 + 更深度的维护者协作 |

## 赞助报告

每月共享的指标包括：

- npm 下载量（`ecc-universal`、`ecc-agentshield`）
- 仓库采用情况（stars、forks、贡献者）
- GitHub App 安装趋势
- 发布节奏和可靠性里程碑

如需获取具体的命令片段和可重复的拉取流程，请参阅 [`docs/business/metrics-and-sponsorship.md`](docs/business/metrics-and-sponsorship.md)。

## 期望与范围

- 赞助用于支持项目维护和加速开发，不涉及项目所有权转让。
- 功能请求的优先级将根据赞助层级、生态系统影响和维护风险综合评估。
- 安全和可靠性修复优先级高于新功能开发。

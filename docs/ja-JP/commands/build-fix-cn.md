# 构建修复

逐步修复 TypeScript 和构建错误：

1. 执行构建：npm run build 或 pnpm build

2. 解析错误输出：
   * 按文件分组
   * 按严重程度排序

3. 对于每个错误：
   * 显示错误上下文（前后 5 行）
   * 说明问题
   * 提出修复方案
   * 应用修复
   * 重新执行构建
   * 确认错误是否已解决

4. 在以下情况下停止：
   * 修复导致新错误
   * 相同错误在 3 次尝试后仍然存在
   * 用户请求暂停

5. 显示摘要：
   * 已修复的错误
   * 剩余的错误
   * 新引入的错误

为安全起见，一次只修复一个错误！

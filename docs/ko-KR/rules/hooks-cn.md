# 钩子系统

## 钩子类型

- **PreToolUse**: 工具执行前（有效性验证，参数修改）
- **PostToolUse**: 工具执行后（自动格式化，检查）
- **Stop**: 会话结束时（最终验证）

## 自动接受权限

小心使用：
- 仅在可信的、定义良好的计划中激活
- 在探索性工作中禁用
- 绝对不要使用 dangerously-skip-permissions 标志
- 而是在 `~/.claude.json` 中设置 `allowedTools`

## TodoWrite 最佳实践

使用 TodoWrite 工具：
- 跟踪多步骤任务的进度
- 验证对指令的理解程度
- 可实时调整方向
- 显示详细实施步骤

通过 Todo 列表可以确认：
- 顺序不正确的步骤
- 遗漏的项目
- 不必要的额外项目
- 错误的细化水平
- 被误解的需求
